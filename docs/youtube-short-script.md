# JustQuests — YouTube Short script (v0.2.0 showcase)

Everything you need to record + edit a ~60-second vertical Short. Spoken
lines are in English (widest reach for a mod); swap to German if you prefer
— timings still work. `[pause]` = leave a short beat for editing/breathing.

## Production specs
- **Format:** vertical **9:16**, 1080×1920, ~60 s (YouTube Short).
- **Capture Minecraft at 16:9, then crop/zoom to 9:16** centered on the
  quest book — or record the GUI windowed and place it in a 9:16 frame.
- **Captions:** burn in big subtitles (most Shorts are watched muted).
- **Pace:** fast cuts, one idea per shot. Music: upbeat, low volume under VO.
- **Record the GUI in singleplayer** (that's where it runs) on the test
  instance (1.21.1), key **J** to open it.
