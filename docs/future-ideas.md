# Future Ideas

Collection point for JustQuests ideas that are **not** scheduled for any
release. New ideas get added here over time — as a short section in this
file, or as their own `docs/idea-*.md` file with a link from here.

> Rule for assistants and contributors: nothing in this file is a work
> order. Do not implement anything from here unless it is explicitly moved
> into the roadmap (README) first.

## How to add an idea

1. One or two sentences in the table above, **or** a dedicated
   `docs/idea-<name>.md` if it needs more room
2. Date it
3. Keep it out of the README roadmap until it is actually planned
